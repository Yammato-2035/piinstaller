# Rescue Remote — Development Control Center (EN)

## Tab "Rescue Remote" (phase 1 design)

Display paired agents, heartbeat, capabilities, allowlisted runbooks, job history, redacted log export, disconnect.

Allowed actions: readonly/diagnostic runbooks only (see `RESCUE_REMOTE_RUNBOOK_JOB_MODEL.md`).

Forbidden UI: shell, arbitrary command, restore, backup, USB write, mount RW, format, apt.
