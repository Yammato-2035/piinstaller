# Telemetry Privacy FAQ (EN)

## Why telemetry?
Improve driver/firmware detection and issue classification on the rescue stick.

## What is sent?
Only schema-valid, redacted payloads after consent and stick verification.

## Revocation?
Consent can be revoked in the beta portal; quarantine data is deleted after retention.

## Remote commands?
**Never.** The telemetry server does not command the stick.
