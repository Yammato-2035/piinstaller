# FAQ — Rettungsstick NTFS/Windows

**F: Kann der Stick mein Windows-Passwort zurücksetzen?**  
Nein. Das ist ausdrücklich verboten.

**F: BitLocker — werden meine Dateien lesbar?**  
Nur mit Recovery-Key. Ohne Key: höchstens Rohimage/Strukturprüfung, keine Entschlüsselung.

**F: Muss ich mich nach Restore anmelden können?**  
Nein, wenn das Passwort fehlt. Abnahme = Bootstruktur plausibel bis Login/Lockscreen.

**F: Wird auf NTFS geschrieben?**  
In F.1/F.2-Vorbereitung: nein. Schreiben nur mit gesonderter Freigabe.

**F: Welches Laufwerk ist die Windows-Quelle?**  
Muss per Precheck eindeutig sein — nie automatisch raten.
