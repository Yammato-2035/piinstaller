# Produkt-Testbadges

| Badge | Bedeutung | Voraussetzung für Vergabe |
|-------|-----------|---------------------------|
| Setuphelfer getestet | Kern-Team hat mit definierter Version und Evidence getestet | Evidence-JSON + Matrix-Eintrag Grün |
| Community bestätigt | mehrere unabhängige Rückmeldungen | dokumentiert, kein Ersatz für internes Testen |
| Daten geprüft | Spezifikationen/Datenblätter verifiziert | Quelle angegeben |
| Budget-Tipp | preislich attraktiv | klar als Meinung/Empfehlung, nicht „beste Qualität“ |
| ungeprüfter Partnerlink | nur Transparenz | keine Qualitätsaussage |
| nicht empfohlen | absichtlich | Begründung (z. B. Sicherheit, Support) |

## Verbot

- Badge „Setuphelfer getestet“ **ohne** Hardware-/Software-Evidence.  
- Vermischung von „geprüft“ und „Provision hoch“.
