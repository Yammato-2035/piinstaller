# Manual Runtime Failure Test Sessions

Konkrete manuelle Testsessions aus den Failure-Operator-Checklisten.

- Nur `manual_only`, keine automatische Ausführung
- Keine destruktiven Schreiboperationen; keine Auto-Reparatur/Freigabe/Ingestion/Release

API: `POST /api/deploy/runner/manual-runtime/failure-test-sessions`
