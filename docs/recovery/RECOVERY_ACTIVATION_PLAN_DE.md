# Recovery Activation Plan (DE)

## Bedeutung
Activation beschreibt einen sicheren Aktivierungsplan, um ein Recovery-Ziel später erreichbar zu machen.

## Wichtiger Rahmen
- keine Aktivierung in dieser Phase
- keine Port-Öffnung durch Ausführung
- keine Dienststarts
- nur strukturierte Schritte + Risiken + Blockierungen
