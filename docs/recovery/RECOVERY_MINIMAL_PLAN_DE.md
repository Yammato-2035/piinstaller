# Recovery Minimal Plan (DE)

## Ziel
Erzeugt einen reinen Plan für ein minimales Recovery-System (SSH + Setuphelfer-Agent/Backend) ohne Ausführung.

## Eigenschaften
- nur Analyse + Vorschläge
- keine Installation
- keine Netzwerkanpassung
- keine Aktivierung von Diensten

## API
`POST /api/recovery/minimal/plan`
