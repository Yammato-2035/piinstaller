# KB: DEPLOY_RUNNER_RELEASE_READINESS

## Ueberblick

Read-only Matrix fuer den Freigabestand aller Runner-Bausteine inklusive Risiko- und Gap-Bewertung.

## Kernpunkte

- Alle Pflichtkomponenten in einer einheitlichen Matrix
- Blockierende und nicht-blockierende Gaps separat
- Keine Produktionsfreigabe in dieser Phase

## API

- `POST /api/deploy/runner/release/readiness`
