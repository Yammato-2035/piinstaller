# KB: DEPLOY_RUNNER_LAB_READINESS_PLAN

## Ueberblick

Read-only Plan zur strukturierten Abarbeitung blockierender Lab-Gaps aus der Release-Readiness.

## Leitprinzip

Alle Schritte sind manuell (`manual_operator_required=true`) und nicht automatisch (`auto_allowed=false`).

## API

- `POST /api/deploy/runner/lab-readiness/unblock-plan`
