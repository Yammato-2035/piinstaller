# KB: DEPLOY_RUNNER_MANUAL_RUNTIME_RESULT_EDIT_CHECKER

## Ueberblick

Read-only Vorpruefung fuer manuell ausgefuellte Runtime-Result-Dateien.

## Warum

Operatoren sehen vor der strikten Ingestion frueh, welche Felder fehlen, welche Risiken blockieren und welcher Validator-Status erwartbar ist.

## API

- `POST /api/deploy/runner/manual-runtime/result-check`
