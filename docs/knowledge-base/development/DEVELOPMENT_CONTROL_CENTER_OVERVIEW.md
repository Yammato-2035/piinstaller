# Development Control Center Overview

Central read-only dashboard for Setuphelfer development governance.

## Components

- Tabbed UI in `ExternalDevelopmentControlCenter`
- Summary API: `/api/dev-dashboard/control-center-summary`
- Roadmap registry: `docs/roadmap/setuphelfer_roadmap.json`

## Safety

No write actions from the control center. ISO build status remains pending until evidence exists.

See also: `docs/architecture/DEVELOPMENT_CONTROL_CENTER_INFORMATION_ARCHITECTURE.md`
