# Rescue Developer Edition — Local Lab Profile

This profile enables the Setuphelfer Development Agent for **local lab telemetry only**.

- **Not** used for public rescue ISO builds
- **No** ISO build in this directory — profile files only
- Agent sends read-only reports to a local Development Server (`127.0.0.1:8000`)

See: `docs/architecture/RESCUE_DEVELOPER_PROFILE_AGENT_CONTRACT.md`
