Boot-Probleme

Symptom: Das System startet nicht oder bleibt beim Bootvorgang stehen.

{{LUCIDE:power:72}}

Moegliche Ursachen

  • Defektes/inkompatibles Netzteil
  • Beschaedigte SD-Karte
  • Image fehlerhaft geschrieben
  • Boot-Konfiguration fehlerhaft

Diagnose

  1. LED-Verhalten am Geraet pruefen
  2. Anderes Netzteil testen
  3. SD-Karte auf anderem System pruefen
  4. Image neu schreiben und erneut testen

Loesung Schritt fuer Schritt

  1. Mit bekannt stabilem Netzteil starten.
  2. SD-Karte neu flashen (sauberes Image).
  3. Nur Basis-Hardware anschliessen, USB-Geraete trennen.
  4. Nach erfolgreichem Boot schrittweise erweitern.

Wann abbrechen

Wenn das System trotz neuem Netzteil und neuer SD-Karte weiterhin nicht startet.

Wann Community fragen

Wenn LED-Codes unklar bleiben oder nur bestimmte Hardwarekombinationen Probleme machen. Zur Community
