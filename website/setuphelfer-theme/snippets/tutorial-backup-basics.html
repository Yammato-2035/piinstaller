Software und Standards

Verwendet u. a.: rsync, OpenSSH (Remote-Kopien), Samba wenn das Ziel eine SMB-Freigabe ist.

  • rsync
  • OpenSSH
  • Samba

Voraussetzungen

  • Externer Speicher oder NAS
  • Zu sichernde Ordner definiert
  • Regelmaessiger Zeitplan

Schritte

  1. Zu sichernde Datenbereiche festlegen
  2. Backup-Ziel einrichten
  3. Erste Sicherung ausfuehren
  4. Wiederherstellungstest machen

Hinweise

  • Backup nicht auf demselben Datentraeger wie das Live-System
  • Regelmaessig pruefen, ob Sicherungen noch lesbar sind

Typische Fehler

  • Nur manuelle Backups ohne Routine
  • Nie getestete Wiederherstellung
  • Unklare Ordnerstruktur im Backup
