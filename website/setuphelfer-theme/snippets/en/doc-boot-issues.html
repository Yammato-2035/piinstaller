Troubleshooting

Boot problems

Symptom: the system does not start or hangs during boot.

{{LUCIDE:power:72}}

Possible causes

  • Faulty or incompatible power supply
  • Damaged SD card
  • Image written incorrectly
  • Boot configuration error

Diagnosis

  1. Check LED behaviour on the device
  2. Try another power supply
  3. Test the SD card on another system
  4. Re-flash the image and test again

Step-by-step fix

  1. Boot with a known stable power supply.
  2. Re-flash the SD card (clean image).
  3. Connect only core hardware; disconnect USB devices.
  4. After a successful boot, add peripherals gradually.

When to stop

If the system still will not boot after a new power supply and SD card.

When to ask the community

If LED patterns stay unclear or only certain hardware combinations fail. Community
