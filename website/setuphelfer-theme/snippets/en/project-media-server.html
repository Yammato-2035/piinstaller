Raspberry Pi as media server

Build your own server for movies, shows and music.

Beginner90 minutes

Benefit

One central library for several devices and users on the home network.

Hardware

  • Raspberry Pi
  • Storage (SD or NVMe)
  • Network

Software and standards

Uses e.g.: Docker (often for media containers), Samba (SMB shares for clients), Plex or Jellyfin as media backend, OpenSSH for secure remote maintenance.

Official projects and documentation:

  • Jellyfin · GitHub
  • Plex
  • Docker Engine docs · Moby (GitHub)
  • Samba
  • OpenSSH · OpenSSH (GitHub)

Typical issues
