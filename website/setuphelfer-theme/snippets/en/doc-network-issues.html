Troubleshooting

Network issues

Symptom: the system is unreachable or DNS/hostname fails.

{{LUCIDE:network:72}}

Possible causes

  • IP conflict or wrong gateway
  • Misconfigured DNS servers
  • Service running but port blocked
  • Unstable Wi‑Fi

Diagnosis

  1. Check IP and hostname
  2. Ping router and target host
  3. Test DNS resolution separately
  4. Check service status and open ports

Step-by-step fix

  1. Clean up static/fixed addressing.
  2. Correct DNS and gateway.
  3. Set firewall rules for required ports.
  4. Retest connectivity after each step.

When to stop

If network parameters are unclear and several services are affected at once.

When to ask the community

For complex router/VLAN setups or recurring disconnects. Community
