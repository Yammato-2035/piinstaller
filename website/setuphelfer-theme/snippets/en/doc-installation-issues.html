Troubleshooting

Installation issues

Symptom: setup or package install fails; services do not start.

{{LUCIDE:download:72}}

Possible causes

  • Missing dependencies
  • Outdated package sources
  • Insufficient disk space
  • Service configuration error

Diagnosis

  1. Read package manager output carefully
  2. Check free disk space
  3. Compare service status and logs
  4. Review recently changed configuration

Step-by-step fix

  1. Refresh package sources and retry.
  2. Install missing dependencies explicitly.
  3. Revert broken config to the last working version.
  4. Restart the service and verify status.

When to stop

If several unrelated errors appear and the root cause is no longer clear.

When to ask the community

If errors stay reproducible after basic checks. Community
