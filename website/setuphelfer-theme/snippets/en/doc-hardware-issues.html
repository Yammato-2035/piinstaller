Troubleshooting

Hardware issues

Symptom: instability, crashes or temperature problems under load.

{{LUCIDE:wrench:72}}

Possible causes

  • Undersized power supply
  • Failing storage
  • Heat buildup without cooling
  • Unstable USB devices

Diagnosis

  1. Check voltage warnings and kernel logs
  2. Measure temperature under load
  3. Test storage with a diagnostic run
  4. Disconnect USB devices one by one and retest

Step-by-step fix

  1. Use a stable power supply.
  2. Replace faulty storage and reinstall.
  3. Improve cooling (airflow/heatsink).
  4. Keep only necessary USB devices connected permanently.

When to stop

If random crashes continue after replacing power and storage.

When to ask the community

For hard-to-reproduce load issues or mixed setups with many add-ons. Community
