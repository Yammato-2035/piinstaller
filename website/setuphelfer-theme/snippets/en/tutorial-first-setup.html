First Raspberry Pi setup

Goal: a stable start with working network, safe credentials and an up-to-date base.

Beginner20–30 minutes
{{LUCIDE:list-ordered:72}}

Software and standards

Uses e.g.: Raspberry Pi Imager / OS image, apt for updates, OpenSSH for remote access, home network (NetworkManager or wpa_supplicant).

  • Raspberry Pi Imager
  • Apt
  • OpenSSH

Requirements

  • Raspberry Pi + power supply + SD card
  • Ethernet or Wi‑Fi
  • Monitor/keyboard or SSH access

Steps

  1. Write image and complete first boot
  2. Set hostname and credentials
  3. Update the system
  4. Test network and access
