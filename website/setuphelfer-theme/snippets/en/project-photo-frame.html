Project

Digital photo frame

Show photos automatically and run the Pi cleanly as a display system.

Beginner30–60 minPi 4/5
{{LUCIDE:image:72}}

Benefit

A stable photo display at home with low maintenance.

Requirements

  • Display or monitor
  • Image folder local or on network
  • Clean boot configuration

Software and standards

Uses e.g.: OpenSSH (maintenance), rsync to sync new photos, Samba if the image folder is on a network share.

  • OpenSSH
  • rsync
  • Samba

Step by step

  1. Set up system and test screen.
  2. Define image source and folder layout.
  3. Install viewer and set fullscreen.
  4. Enable autostart on boot.
  5. Check power and reboot behaviour.

Typical issues

  • Black screen after standby
  • Wrong image path in start script
  • Network shares without stable connection

Next steps

  • Automatic image refresh
  • Schedule on/off
  • Set up remote maintenance
