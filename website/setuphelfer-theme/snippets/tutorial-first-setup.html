Erste Raspberry-Pi-Einrichtung

Ziel: Ein stabiler Start mit funktionierendem Netzwerk, sicheren Zugangsdaten und aktueller Basis.

Anfaenger20-30 Minuten

Software und Standards

Verwendet u. a.: Raspberry Pi Imager / OS-Image, apt fuer Updates, OpenSSH fuer Fernzugriff, Heimnetz (NetworkManager oder wpa_supplicant).

  • Raspberry Pi Imager
  • Apt
  • OpenSSH

Voraussetzungen

  • Raspberry Pi + Netzteil + SD-Karte
  • Netzwerkkabel oder WLAN-Zugang
  • Monitor/Tastatur oder SSH-Zugang

Schritte

  1. Image schreiben und Erststart durchfuehren
  2. Hostname und Zugangsdaten setzen
  3. System aktualisieren
  4. Netzwerk und Zugriff testen
