
SetupHelfer WordPress Theme
==========================

Installation
------------
1. ZIP unter Design > Themes > Theme hochladen installieren.
2. Theme aktivieren.
3. Nach Aktivierung erzeugt das Theme automatisch:
   - die statische Startseite
   - die Seiten Community, Download und Über SetupHelfer
   - Beispielinhalte für Projekte, Tutorials und Dokumentation
4. Permalinks einmal unter Einstellungen > Permalinks speichern.

Hinweise
--------
- Projekte, Tutorials und Dokumentation laufen als Custom Post Types.
- Das Theme nutzt vorbereitete Inhalte und Assets aus der statischen SetupHelfer-Basis.
- Für ein Forum kann bbPress später zusätzlich installiert werden.
