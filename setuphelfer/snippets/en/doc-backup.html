Backup

Backups are often planned too late. This page defines a minimal standard before projects go live.

{{LUCIDE:hard-drive:72}}
  • Treat configuration and data separately
  • Prefer external backup targets
  • Plan restore paths, not only copies
  • Do not confuse snapshots with full backups
{{LUCIDE:clipboard-list}}

Practical rule

Change only as much at once as you can trace and roll back if needed.
