Mail server

Productive mail needs far more care than many beginners expect. This topic is clearly marked as advanced.

{{LUCIDE:mail:72}}
  • DNS, SPF, DKIM and DMARC are mandatory
  • No production use without delivery tests
  • Use small test systems for experiments only
  • Keep web platform and mail operation separate where possible

Practical rule

Change only as much at once as you can trace and roll back if needed.
