Diagnostics

Troubleshooting should be structured: check service status, network, storage and logs separately — not everything at once.

{{LUCIDE:search:72}}
  • Name the symptom clearly
  • Check service status
  • Test network and name resolution separately
  • Read log files only as far as they match the question

Practical rule

Change only as much at once as you can trace and roll back if needed.
