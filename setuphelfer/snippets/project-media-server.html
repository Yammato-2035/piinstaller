Projekt

Raspberry Pi als Medienserver

Baue deinen eigenen Medienserver fuer Filme, Serien und Musik.

Anfaenger90 Minuten

Nutzen

Eine zentrale Bibliothek fuer mehrere Geraete und Nutzer im Heimnetz.

Hardware

  • Raspberry Pi
  • Speicher (SD oder NVMe)
  • Netzwerk

Software und Standards

Verwendet u. a.: Docker (haeufig fuer Medien-Container), Samba (SMB-Freigaben fuer Clients), Plex oder Jellyfin als Medienbackend, OpenSSH fuer sichere Fernwartung.

Offizielle Projekte und Dokumentation:

  • Jellyfin — offizielle Website · GitHub
  • Plex
  • Docker Engine — Dokumentation · Moby (GitHub)
  • Samba
  • OpenSSH · OpenSSH (GitHub)

Schritt-fuer-Schritt

  1. System installieren.
  2. Software einrichten.
  3. Medien hinzufuegen.
  4. Zugriff konfigurieren.

Typische Fehler

  • Netzwerk nicht erreichbar
  • Falsche Freigaben
  • Langsamer Speicher

Naechste Schritte

  • Automatisches Backup einrichten
  • Monitoring fuer Speicher und Dienststatus
  • Optional: Remote-Zugriff sicher konfigurieren
