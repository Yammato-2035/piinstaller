# Setuphelfer – Projekt-Roadmap

## Aktuelle Arbeitsreihenfolge
1. Backup / Restore härten
2. Bluetooth / Peripherie sauber ausbauen
3. Raspberry-Pi-Konfiguration konsolidieren
4. Pi-hole integrieren
5. Lernpfad GPIO / Sensorik / Freenove ausbauen
6. Diagnosemodul entwickeln
7. Spätere Erweiterungen: Retro PI, KI-Kurs, Cluster, Live-Verschlüsselung

## Arbeitsprinzip
- Nur ein Abschnitt gleichzeitig
- Jeder Abschnitt endet mit Tests, Logeintrag und Review
- Aus Problemen wird strukturiertes Wissen für FAQ, Tutorials und Diagnose abgeleitet
